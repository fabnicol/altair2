
                        ALTAIR

            Analyse des lignes de traitement, 
   attributions indemnitaires et autres rémunérations


Le présent logiciel produit automatiquement des statistiques et des tests
de cohérence et de conformité comptable et réglementaire, à partir de bases
de données issues de l'application Xémélios (Direction générale des finances 
publiques, http://xemelios.org).

Exécution
---------

1. Lancer RStudio

  Cliquer sur l'icône "R" bleue ou lancer rstudio.exe dans RStudio\bin

2. Dans l'interface RStudio

  Vérifier que l'onglet actif est bien le fichier 'exemple.R'. Si ce script
  ne s'affiche pas il est possible de l'afficher en cliquant sur l'onglet 
  Fichier de la fenêtre inférieure droite, sous le dossier Tests\Exemple.

  Puis, appuyer sur l'icône "carnet de notes" à la droite de l'icône 
  "baguette magique" et la gauche de 'Run', ou cliquer sur le menu
  Code > Source, ou appluyer simultanément sur Ctrl + Majuscule + S

3. Choisir le type de rapport (.html, .pdf, ou MS Word) dans le dialogue de
   l'interface graphique.

   Le rapport d'exécution est généré dans le dossier Tests\Exemple.

   Les bases de données sont produites dans le sous-dossier Exemple\Bases.

Exemple de production   
---------------------

Voir sous-dossier Tests\Exemple\Rapport ainsi que le dossier des bases
.csv produites en sortie de l'application.

Il est possible de visualiser sous Excel ou Wordpad les bases liées par
des liens hypertextes dans les rapports (.pdf, .docx, .html)
