Pour la plateforme de développement penser à ajouter à la main les deux chemins suivants au PATH :

$ALTAIR/Git/bin

où $ALTAIR est le présent dossier.
Ces chemins sont automatiquement installés par l'installateur pour l'installation complète.

Pour la création de l'installateur Windows :

a) récupérer Local\RStudio-Desktop et Roaming\RStudio dans AppData et les placer dans les dossiers
respectifs locaux

b) Récupérer le dossier .Rproj.user et le placer dans le dossier RStudio-project

Attention :  avant d'avori fait cela, c'est l'application utilisateur qui doit avoir été lancée sous C:\Users\Public\Altair
