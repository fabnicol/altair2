Actualisation de la distribution
--------------------------------
Suivre les séquences suivantes dans l'ordre :

1. Dans le répertoire C:\Users\Public\Altair, double-cliquer sur le fichier git_altaïr.bat

2. Suivre les demandes de la console (entrer le login etc.)
   Une connexion internet est requise. Le proxy est pris en compte.

3. La version actualisée se télécharge (environ 20 minutes) dans C:\Users\Public\altair.git

4. Archiver l'ancienne version en renommant l'ancien répertoire C:\Users\Public\Altair
  (par exemple C:\Users\Public\Altair.arch). En cas de problème il pourra toujours être restauré.
  Il est aussi possible de le zipper puis de l'effacer.

5. Renommer manuellement C:\Users\Public\altair.git comme : C:\Users\Public\Altair

6. Dans le nouveau répertoire C:\Users\Public\Altair, double-cliquer sur le fichier 
   finaliser_distribution.bat

   La version est alors actualisée et peut s'utiliser comme la version de distribution.

Actualisations ultérieures
--------------------------

Dans l'interface RStudio, sélectionner l'onglet Git dans le quadrant supérieur droit.
Cliquer sur l'icône "Pull" une ou deux fois par mois.
Le code et l'exécutable sont alors automatiquement actualisés.

Avertissement
------------- 

Cette distribution actualisée par gestionnaire de versions requiert 4,8 Go d'espace disque 
et ne fonctionne qu'avec des processeurs de niveau minimal corei3.
