Guide d'installation
--------------------

Choisir l'installateur .celeron pour une installation sur portables standard.
Choisir l'installateur .core pour une installation sur portables multicoeurs (core2, corei3 et plus).

1.  Copier l'installateur sur le bureau et l'exécuter.

2.  Choisir le chemin d'installation par défaut c:\Users\Public

3.  Installer les dépendances logicielles R-3.1.2, RStudio et Miktex.

4.  Installation de R-3.1.2 : utiliser les chemins d'installations par défaut.
    Décocher la case "32 bits" lorsque le choix entre plusieurs plateformes (32 bits, 64 bits... est proposé)
	
5.  Installation de RStudio : utiliser les chemins d'installations par défaut.

6.  Attention, lorsqu'un premier installateur dépendant a été exécuté, pour passer au suivant, il faut
    parfois cliquer sur la barre d'état de Windows (en bas de l'écran).
	
7.  Installation de Miktex : attention à choisir les bonnes options.
    a) cocher la case "I accept..." du premier dialogue, en bas à gauche
	  b) puis cocher "Install Miktex" au dialogue suivant
    c) laisser le bouton radio à "Basic Miktex" au dialogue suivant
	  d) laisser le choix de l'utilisateur à "Anyone..." au dialogue suivant
    e) puis préciser le répertoire dans lequel seront prises les feuilles de style :
	   à ce stade il est important de sélectionner (bouton Browse) le répertoire Miktex du DVD
	   
      
8.  Installer vc_redistx64.exe de Microsoft. Ce sont des bibliothèques dynamiques qui permettent d'exécuter 
    les programmes compilés avec le compilateur Microsoft MSVC2013.
   
9.  Exécuter l'interface.
    Configurer l'interface en cochant toutes les cases sauf plein écran.
   
10. Ouvrir un nouveau projet (Fichiers > Nouveau projet)
    Importer dans le projet un des deux répertoires :
   
       Tests\Exemple\Donnees\xhl\Anonyme
    ou Tests\Exemple\Donnees\xhl\Anonyme2
   
11. Ouvrir le dialogue des paramètres. 
    Renseigner le répertoire de sortie de la base .csv produite par Altair.
    A ce stade, le répertoire doit être :
   
       Tests\Exemple\Donnees\R-Altair   
   
    Vérifier le nombre de fils d'exécution dans le menu déroulant en bas du dialogue.
    Sur un portable récent il est possible de sélectionner "2" comme nombre de fils d'exécution pour accélérer 
    le traitement. Sur un portable performant (corei5 et plus) , il est possible de sélectionner jusqu'à 
    "6" ou "7" fils d'exécution. Pour exécuter en fond de tâche au contraire, laisser sur "1".
   
    Valider.
   
12. Cliquer sur le bouton "csv" de la barre d'outils.    
    Attendre une à trois minutes.
   
13. Lorsque la barre de progression est à 100 % et que le dialogue des messages confirme "Terminé..."
    appuyer sur le bouton bleu "A" de la barre d'outils, qui lance l'interface RStudio.

    Aller dans l'arborescence "Files"" dans le quart inférieur droit de l'interface et sélectionner le projet
    altair.Rproj.

    Changer les paramètres d'édition de code par défaut : Tools > Global options > Default text encoding > Change > UTF-8
         Tools > Global options > Default text encoding > Change > UTF-8

   
14. Vérifier que l'on a dans le quart supérieur gauche de l'interface un script altair.R. Sinon aller le 
    chercher en cliquant sur le fichier altair.R dans l'arborescence des fichiers et répertoires dans le quart 
  	inférieur droit (onglet Files). Le fichier est localisé sous le répetoire Tests\Exemple.
   
    Appuyer sur le bouton qui ressemble à un cahier ("Notebook") dans la barre d'outils. Vers la fin de 
	  l'exécution du script une série de dialogues apparaît qui demandent d'importer des feuilles de style.
	  Sélectionner à la première fois le répertoire du DVD Miktex-...  dans le menu déroulant.
   
15. Lorsque le pdf apparaît, l'installation est terminée. Veiller à refermer RStudio en laissant le script altair.R 
    sélectionné dans la partie supérieure gauche.
