Lance le décodage des variables Bulletins de paye (Nom, Prenom, Matricule,...)

Paramètres
curpointeur XML courant
infotable d'informations
Renvoie
nombre de lignes de paye

Commence par atteindre le noeud Agent. Vérifie son identification (Nom, Prenom, Matricule, NIR, NbEnfants, Statut, EmploiMetier, Grade, Echelon, Indice) Décode ensuite Evenement, Service, NBI, QuotiteTrav. Appelle ensuite la fonction lignePaye qui décode les lignes de paye de la balise Remuneration. Décode ensuite les champs de fin de fichier : NbHeureTotal, NbHeureSup, MtBrut, MtNet, MtNetAPayer.
