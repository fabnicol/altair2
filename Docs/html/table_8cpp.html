◆ boucle_ecriture()

pair<uint64_t, uint32_t> boucle_ecriture ( vector< info_t > &  Info,
int  nsegment 
)

Lance l'écriture des fichiers Bulletins.paie.csv et Table....csv
pour un segment des données de paye, et à partir de l'ensemble des données de paye décodées.

Paramètres
InfoVecteur de l'ensemble des données info_t de paye décodées (un membre du vecteur par fil d'exécution).
nsegmentRang du segment dans l'ensemble des segments de données de paye.
Note
Le comportament de cette fonction est fortement dépendant des valeurs de :
  • Info[0].generer_rang (-rank)
  • Info[0].separateur (x dans -s x)
  • Info[0].taille_base (x numérique dans -T x)
  • Info[0].type_base (x dans -T x si x est dans {"X", "AN", "A", "AC", "AV", "C", "D", "I", "IR", "RE", "S", "T"}
  • Info[0].select_echelon (-E)
  • Info[0].select_siret (-S)

Le rang de la ligne sera généré en première colonne si -rank est dans la ligne de commande
Le séparateur CSV est donné par -s
La taille de la base induit les découpages de base en morceaux de taille fixe
Le type de la base induit une limitation sur la catégorie de lignes de paye exportées
L'échelon et l'ensemble Budget-Employeur-Siret-Etablissement sont des colonnes optionnelles
Renvoie
Paire du nombre de lignes de paye et du nombre de ligne de bulletins exportés.

Ecrit une ligne de bulletin de paye pour un agent donné dans le fichier CSV Bulletins.paie
.

Paramètres
iRang du fil d'exécution
agentRang de l'agent
bulletinsBase de type ofstream permettant d'écrire le fichier CSV
sepSéparateur de champs CSV (usuellement ';')
InfoVecteur de structures info_t contenant les bases de paye décodées
rangInutilisé
Note
les variables communes à tous les bulletins de paye sont :
  • Nom : Nom de l'agent
  • Prenom : Prénom de l'agent
  • Matricule : Matricule de l'agent
  • Service : Service de l'agent
  • NbEnfants : Nombre d'enfants de l'agent
  • Statut : Statut de l'agent (TITULAIRE, NON_TITULAIRE, STAGIAIRE, EMPLOI_AIDE, EMPLOI_FONCTIONNEL....)
  • QuotiteTrav : Quotité admnistrative (sur 100)
  • NbHeureSup : Nombre d'heures supplémentaires
  • NbHeureTotal : Nombre d'heures total
  • Indice : Indice
  • MtBrut : Montant brut
  • MtNet : Montant net
  • MtNetAPayer : Montant net à payer
  • NBI : Nouvelle bonification indiciaire
  • EmploiMetier : Libellé de l'emploi
  • Grade : Libellé du grade
  • Code : Code de l'événement de paye
  • Description : Description de l'événement de paye
