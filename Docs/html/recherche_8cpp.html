Description détaillée

Ce fichier contient le code permettant de rechercher en mémoire, après décodage des bases XML, un bulletin de paye particulier correspondant à un matricule, un mois et une année. Il contient aussi le code permettant l'itération de cette fonctionnalité sur des intervalles temporels.
