Description détaillée

Classe de l'onglet Codes, qui permet de saisir manuellement les codes de paye de certaines catégories de lignes de paye.

Les codes différents, pour la même ligne, sont séparés par des points-virgules. Ils doivent être exportés dans le fichier exportpath en cliquant sur la flèche verte de l'onglet.
