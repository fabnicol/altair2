◆ ecrire_log()

void ecrire_log ( const info_t &  info,
ofstream &  log,
int  diff 
)

Produit un journal d'exécution.

Paramètres
infotable d'informations
logfichier
diffdifférence entre les analyseurs C et XML

Contenient les colonnes suivantes, pour chaque ligne de paye :

 Année, Mois, Matricule, Rang global, Rang dans le fichier, Analseur C, Xml

et la différence entre l'analyseur C et Xml.
Sépare les colonnes par la chaine " | ".

Note
Le chemin du journal est donné par #info#chemin_log
