sauv.basesR Documentation

sauv.bases

Description

Sauvegarde d'une base data.table sous forme de fichier CSV

Usage

sauv.bases(chemin.dossier, Latin = convertir.latin, env = .GlobalEnv, ...)

Arguments

chemin.dossier

Chemin du dossier dans lequel la base sera sauvegardée

Latin

(= convertir.latin) Convertir en encodage latin ISO-8859-15

env

(= .GlobalEnv) Environnement

...

Autres noms d'objets à sauvegarder

nom

Nom de l'objet à sauvegarder

Value

Liste de booléens résultant de l'application de Sauv.base

Examples

envir <- environment()
# Générer Bulletins.paie et Paie dans envir
sauv.bases("données", env = envir, c("Bulletins.paie", "Paie"))
