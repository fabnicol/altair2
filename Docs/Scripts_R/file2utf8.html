file2utf8R Documentation

Conversion d'un fichier de ISO-8859-15 en UTF-8

Description

Conversion d'un fichier de ISO-8859-15 en UTF-8

Usage

file2utf8(nom, encodage.in = "ISO-8859-15")

Arguments

nom

Chemin du fichier à encoder

encodage.in

(= encodage.entrée) Encodage du fichier de lecture

Value

Lancement dun appel système à iconv -t UTF-8
