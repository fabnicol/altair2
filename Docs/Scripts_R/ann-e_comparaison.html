année_comparaisonR Documentation

Année de comparaison avec les données nationales.

Description

Calcule l'année à laquelle la pyramide des âges va être comparée aux données nationales, pour un versant donné de la focntion publique.

Usage

année_comparaison(versant)

Arguments

versant

Chaîne de caractères parmi "FPT", "FPH", "TIT_FPT", "TIT_FPH", "NONTIT_FPT", "NONTIT_FPH".

Value

Une liste dont les composantes sont :

année

l'année de comparaison (entier).

pyr

la base de données nationales correspondante (data.table)
