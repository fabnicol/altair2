       
              PROCEDURE D'ANONYMISATION version 3  (janvier 2016)


Pour anonymiser des bases xhl :

1. Placer les bases de paye (.xhl) comme vous l'entendez dans un répertoire de travail,
   disons 'Travail' par exemple (le nom est libre). 
   Il est conseillé d'inclure des bases de paye dans des sous-répertoires, dont le nom est libre.
   Vérifier que toutes les bases de données ont l'extension .xhl

2. Copier les fichiers extraits de 'Anonymiser v3.zip' à la racine de ce répertoire de travail. 
   Ce répertoire de travail doit à présent être votre répertoire courant.

3. Cliquer sur Anonymiser.sh. Si une console ne s'ouvre pas avec des messages d'exécution, alors :
   a) cliquer sur le fichier bash.exe
   b) dans la console, taper ./Anonymiser.sh
   c) taper "Entrée"

4. Attendre que le dialogue de la console qui s'ouvre arrive à son terme.

5. Zipper le(s) dossier(s) contenant les bases et envoyer.


Cette nouvelle procédure d'anonymisation est à la fois plus sûre, plus souple et trois à quatre fois 
plus rapide que les versions antérieures. En cas d'erreurs de conversion consulter le fichier 
log.erreurs et le communiquer pour diagnostic.


Mode d'exécution alternatif (ancienne version)
___________________________

En cas de problème avec la procédure précédente, le mode d'exécution alternatif de la version précédente
du logiciel peut être utilisé en guise de méthode de secours.

La configuration à respecter est toutefois plus stricte.

Pour anonymiser des bases xhl :

1. Placer les bases dans un ou plusieurs répertoires contigus comme sur l'image jointe.
   Le nom des répertoires est libre.
   Il ne doit pas y avoir de sous-répertoire au sein de ces répertoires.
   L'ensemble du chemin des noms de fichier doit être dénué d'espace.

2. Placer ce ou ces répertoires ainsi créés dans un même répertoire vide, disons Nouveau dossier.
   Copier les fichiers extraits de Anonymisezr.zip dans ce répertoire Nouveau dossier.

3. Cliquer sur Anonymiser_alternative.sh. Si une console ne s'ouvre pas avec des messages d'exécution, alors :
   a) cliquer sur le fichier bash.exe
   b) dans la console, taper ./Anonymiser_alternative.sh
   c) taper "Entrée"

4. Attendre que le dialogue de la console qui s'ouvre arrive à son terme.

5. Zipper le(s) dossier(s) contenant les bases et envoyer.
