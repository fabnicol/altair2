
                        Altaïr-SDK
                       ___________

            Analyse des lignes de traitement, 
   attributions indemnitaires et rémunérations diverses
        Environnement de développement logiciel


L'ensemble des logiciels installés concourt au développement du logiciel
Altaïr, qui produit automatiquement des statistiques et des tests
de cohérence et de conformité comptable et réglementaire, à partir de bases
de données issues de l'application Xémélios (Direction générale des finances 
publiques, http://xemelios.org).


Liste des logiciels installés
_____________________________

mingw64-5.2 : compilateur dérivé du compilateur GCC version 5.2 pour Windows

msys64 :      environnement d'exécution et de compilation sous Windows 
              de code source compatible unix

NSIS :        logiciel de compilation de l'installateur

Notepad++ :   éditeur de fichiers xml et de code source

qt-5.5.0-x64-mingw510r0-seh-rev0 : bibliothèques Qt5.5 précompilées pour compilation
              de code source sous Windows avec le compilateur mingw64-5.2. Les exécutables
              produits sont liés dynamiquement aux bibliothèques. Utilisation conseillée
              pour le débogage.

qt-5.5.0-x64-mingw52-static-runtime : les mêmes bibilothèques mais pour produire des 
              exécutables compilés statiquement. Utilisation conseillée pour le déploiement.

qtcreator-3.5.0 : environnement de développement intégré (IDE) pour la compilation et 
              le déploiement de logiciels utilisant les bibliothèques Qt5

Rtools :      plateforme de compilation de bibliothèques pour le langage R

redist.exe :  bibliothèques d'exécution Microsoft 


Utilisation de base
___________________

Accepter au moins les installations des logiciels suivants dans une configuration minimale :

  qtcreator-3.5.0
  mingw64-5.2
  qt-5.5.0-x64-mingw52-static-runtime

et accepter la décompression du code source en fin d'exécution.

Cliquer sur l'icône Altaïr-SDK installée sur le bureau ou dans le menu démarrer.
Après l'ouverture de l'IDE qtcreator, cliquer sur la flèche verte en bas à gauche de l'interface.
L'ensemble du logiciel Altaïr sera compilé et exécuté.

Usage avancé
____________

msys64 peut être utilisé pour lancer des lignes de commande longues (que les consoles Windows
habituelles ne savent pas gérer).

Rtools sera automatiquement détecté par RStudio lors de la compilation de bibliothèques à partir
de code téléchargé (cas de de la bibliothèque data.table).

Notepad++ est très utile, paramétré en affichage de langage XML, pour visualiser les fichiers .xhl 
des bases de paye, ainsi que pour afficher de manière autonome tous les types de code source.

NSIS peut être utilisé, avec une version du fichier plateforme.nsi, pour recompiler le présent 
installateur.

Gestionnaire de versions
------------------------

Dans le dossier altair\Git, une console git (git-bash.exe) permet de recouvrer 
les éléments du dossier altair en cas de perte accidentelle.

Il suffit de lancer la console et d'exécuter :

    git reset --hard master

Attention cette opération effacera toutes les modifications réalisées depuis
l'installation.
